  **Script 1: create_weapons.py** -- Creates three geometric weapon models, each as a separate collection/object hierarchy, then saves to weapons.blend.

  **Rifle** (~0.8m long, 0.05m wide):
  - Body: Elongated box (0.8 x 0.05 x 0.12m), dark grey (#333333)
  - Barrel: Thin cylinder extending forward from body, radius 0.012m, length 0.3m, dark grey
  - Stock: Box at rear (0.15 x 0.04 x 0.08m), dark grey
  - Magazine: Small box below body (0.03 x 0.06 x 0.08m), dark grey
  - Scope rail: Thin box on top (0.2 x 0.02 x 0.015m), dark grey
  - Join all parts into one object named "Rifle", origin at grip position (where hand holds it)

  **Pistol** (~0.22m long, compact):
  - Slide: Box (0.18 x 0.035 x 0.04m), dark grey
  - Grip: Angled box below rear of slide (0.03 x 0.08 x 0.04m), slightly lighter grey
  - Barrel: Short cylinder, radius 0.008m, length 0.05m extending from slide front
  - Join all parts into one object named "Pistol", origin at grip

  **Knife** (~0.28m total):
  - Blade: Flattened box tapered to a point (0.15 x 0.003 x 0.03m), light grey/silver (#AAAAAA)
  - Handle: Cylinder, radius 0.015m, length 0.12m, dark brown (#554433)
  - Guard: Small thin box between blade and handle
  - Join all parts into one object named "Knife", origin at grip

  For each weapon: after joining, apply all transforms, set origin to the grip position (where Hand.R bone will attach). This origin point is where the weapon attaches to the FP arm hand.

  Material: Create a dark gunmetal grey Principled BSDF (0.2, 0.2, 0.2), Roughness 0.7, Metallic 0.3. Knife blade gets a lighter material.

  Save to: `visual/blender/weapons.blend`

  **Script 2: create_fp_arms.py** -- Creates first-person arm model with simplified armature. Only needs arms from shoulders to hands (no head, torso, legs).

  **FP Arms armature** (10 bones):

  Bones positioned for a first-person perspective (arms extending forward from bottom of view). The FP model is NOT at world scale -- it sits in front of the weapon camera at close range.

  **FP Arms mesh:**
  - Upper arms: Cylinders, radius 0.045m, depth 0.25m
  - Forearms: Cylinders, radius 0.038m, depth 0.22m
  - Hands: Small cubes, ~0.05m
  - Wrists: Small spheres at joints, radius 0.03m

  Use the same neutral grey material as the mannequin body (0.6, 0.6, 0.6). Team color applied at runtime.

  Vertex groups matching bone names, Armature modifier on each mesh part, same pattern as the mannequin script.

  Position the arms in a natural "holding weapon" pose: arms extending slightly forward and down, hands together in front at about chest height relative to where the camera would be.

  <action>
  Create TWO export scripts.

  **export_weapons.py** -- Opens weapons.blend and exports each weapon as a separate GLB file. Weapons are static meshes (no armature/skinning needed).

  Use an explicit deselect-all-then-select loop for each weapon to avoid cross-contamination between exports:

  ```python
  import bpy
  import os

  weapon_names = ['Rifle', 'Pistol', 'Knife']

  for name in weapon_names:
      # CRITICAL: deselect ALL objects before selecting the next weapon
      bpy.ops.object.select_all(action='DESELECT')

      # Select only this weapon's object
      obj = bpy.data.objects.get(name)
      if obj is None:
          print(f"WARNING: Object '{name}' not found, skipping")
          continue
      obj.select_set(True)
      bpy.context.view_layer.objects.active = obj

      filepath = os.path.join(os.path.dirname(os.path.dirname(os.path.abspath(__file__))), f'{name.lower()}.glb')
      bpy.ops.export_scene.gltf(
          filepath=filepath,
          export_format='GLB',
          export_skins=False,      # Static mesh, no skeleton
          export_yup=True,
          export_apply=True,
          export_animations=False,
          export_cameras=False,
          export_lights=False,
          use_selection=True,      # Only export selected object
      )
      print(f"Exported: {name} -> {filepath}")
  ```

  If `os.path` construction doesn't work for Blender's CWD, fall back to constructing absolute paths from `bpy.data.filepath` (the .blend file location).

  **export_fp_arms.py** -- Opens fp_arms.blend and exports the rigged arms:
