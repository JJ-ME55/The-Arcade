<objective>
Implement ragdoll physics using Rapier.js that activates when a player mannequin dies, replacing the animated skeleton with physics-driven body parts that collapse naturally.

Purpose: Ragdoll death is a core visual feedback for combat. When a player is eliminated, their body should crumple and tumble based on momentum, not just disappear or play a canned animation. This sells the impact of kills and is a standard feature in competitive FPS games.

Output: RagdollSystem class in src/ragdoll.js, Rapier.js integrated into the project, ragdoll triggerable via keypress for testing (actual death trigger comes in Phase 4).
</objective>

  **RagdollSystem class (src/ragdoll.js):**

  ```javascript
  export class RagdollSystem {
    constructor(THREE) { ... }

    // Initialize Rapier WASM (must await before use)
    async init() { ... }

    // Create ragdoll from a PlayerModel instance's current bone positions
    spawnRagdoll(playerInstance, velocity, scene) { ... }

    // Step Rapier physics world (call each tick at 64Hz)
    step(dt) { ... }

    // Sync visual meshes to physics body positions (call each render frame)
    updateVisuals() { ... }

    // Remove a ragdoll after N seconds
    removeRagdoll(ragdoll) { ... }
  }
  ```

  **init():**
  ```javascript
  import RAPIER from '@dimforge/rapier3d-compat';
  await RAPIER.init();
  this.RAPIER = RAPIER;
  this.world = new RAPIER.World({ x: 0.0, y: -9.81, z: 0.0 });
  this.ragdolls = [];
  ```

  **spawnRagdoll(playerInstance, velocity, scene):**

  1. Hide the SkinnedMesh model: `playerInstance.model.visible = false`
  2. Read bone world positions from the PlayerModel instance's bone map:
     ```javascript
     const bonePositions = {};
     for (const [name, bone] of Object.entries(playerInstance.bones)) {
       const wp = new THREE.Vector3();
       bone.getWorldPosition(wp);
       bonePositions[name] = wp;
     }
     ```
  3. Create rigid bodies and colliders for major body segments (12 bodies):
     - **Head**: sphere collider, radius 0.12, at Head bone position
     - **Torso upper** (Chest): capsule collider, half-height 0.15, radius 0.12, at Chest position
     - **Torso lower** (Spine/Root): capsule collider, half-height 0.12, radius 0.10, at Root position
     - **UpperArm.L/R**: capsule collider, half-height 0.12, radius 0.04
     - **ForeArm.L/R**: capsule collider, half-height 0.11, radius 0.035
     - **Thigh.L/R**: capsule collider, half-height 0.20, radius 0.05
     - **Shin.L/R**: capsule collider, half-height 0.18, radius 0.04

     For each:
     ```javascript
     const desc = RAPIER.RigidBodyDesc.dynamic()
       .setTranslation(pos.x, pos.y, pos.z)
       .setLinvel({ x: velocity.x, y: velocity.y, z: velocity.z })
       .setLinearDamping(0.5)    // Slow down over time
       .setAngularDamping(0.5);  // Reduce spinning
     const body = this.world.createRigidBody(desc);
     const collider = RAPIER.ColliderDesc.capsule(halfHeight, radius)
       .setRestitution(0.1)      // Low bounce
       .setFriction(0.8);        // High friction (don't slide forever)
     this.world.createCollider(collider, body);
     ```

  4. Create joints between connected body parts:
     - **Neck** (Head -> Torso upper): spherical joint
     - **Spine** (Torso upper -> Torso lower): spherical joint
     - **Shoulders** (Torso upper -> UpperArm): spherical joint
     - **Elbows** (UpperArm -> ForeArm): revolute joint (single axis)
     - **Hips** (Torso lower -> Thigh): spherical joint
     - **Knees** (Thigh -> Shin): revolute joint (single axis)

     Spherical joint example:
     ```javascript
     const params = RAPIER.JointData.spherical(
       { x: anchorOnParent.x, y: anchorOnParent.y, z: anchorOnParent.z },
       { x: anchorOnChild.x, y: anchorOnChild.y, z: anchorOnChild.z }
     );
     this.world.createImpulseJoint(params, parentBody, childBody, true);
     ```

     Joint anchors: use half the distance between connected bones, offset from each body's center.

  5. Create visual meshes for each body part (simple primitives matching the mannequin's material):
     - Same shapes as mannequin body parts (cylinders, spheres)
     - Use the same team color material from the mannequin instance
     - Add to scene

  6. Create a ground plane collider in the Rapier world so ragdolls don't fall through the floor:
     ```javascript
     // Only needs to be done once
     if (!this.groundCreated) {
       const groundDesc = RAPIER.ColliderDesc.halfSpace({ x: 0, y: 1, z: 0 });
       this.world.createCollider(groundDesc);
       this.groundCreated = true;
     }
     ```
     NOTE: This is a simplified approach. The ragdoll won't collide with map walls/ramps, only the ground plane. Full map collision for ragdolls is complex and unnecessary -- ragdolls only need to look roughly right for 2-3 seconds.

  7. Store ragdoll in `this.ragdolls` array with: `{ bodies, visuals, joints, scene, spawnTime }`

  **step(dt):**
  ```javascript
  this.world.step();  // Rapier uses its own internal timestep

  // Auto-remove ragdolls after 5 seconds
  const now = performance.now() / 1000;
  this.ragdolls = this.ragdolls.filter(r => {
    if (now - r.spawnTime > 5.0) {
      this.removeRagdoll(r);
      return false;
    }
    return true;
  });
  ```

  **updateVisuals():**
  For each ragdoll, for each body part:
  ```javascript
  const pos = body.translation();
  const rot = body.rotation();
  visual.position.set(pos.x, pos.y, pos.z);
  visual.quaternion.set(rot.x, rot.y, rot.z, rot.w);
  ```

  **removeRagdoll(ragdoll):**
  - Remove visual meshes from scene
  - Remove rigid bodies from Rapier world
  - Remove colliders and joints
